# BattleshipsGame
An implementation of simple version of the Battleships game.
<br>
https://medium.com/guestline-labs/hints-for-our-interview-process-and-code-test-ae647325f400

# How to run
1. Clone the repository
```bash
git clone https://github.com/krzyspie/BattleshipsGame.git
```
2. Build the solution
```bash
dotnet build <path to the folder where is the solution>/BattleshipsGame.sln
```
3. Run the project
```bash
dotnet run --project <path to the folder where is the solution>/Battleships/Battleships.csproj 
```
